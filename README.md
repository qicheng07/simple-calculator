# 简易计算器

一个轻量级的单页Web应用，定位为日常生活场景下的快速计算工具。

## 功能特性

- ✅ 基础四则运算（加、减、乘、除）
- ✅ 小数运算支持
- ✅ 运算优先级（先乘除后加减）
- ✅ 错误处理（除以零、溢出）
- ✅ 响应式界面设计
- ✅ 键盘快捷键支持

## 在线演示

访问：https://qicheng07.github.io/simple-calculator/

## 本地运行

1. 克隆仓库
2. 用浏览器打开 `index.html`
